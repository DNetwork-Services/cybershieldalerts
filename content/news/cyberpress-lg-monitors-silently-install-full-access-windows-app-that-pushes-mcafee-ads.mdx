Plugging in a monitor is not supposed to change the software on your computer. According to testing by CyberDigest, LG monitors do exactly that, triggering a silent installation process that ends with the user staring at a McAfee ad they never asked for. Once an LG UltraGear is connected to a Windows PC, Windows Update automatically fetches and installs the LG Monitor App Installer, with Windows Reliability Monitor logging its arrival roughly a minute after the cable is inserted. There is no consent prompt, no dialog, and no approval step of any kind before the software lands on the machine. The first indication most users get that anything happened is a pop-up advertising a 30-day free trial of McAfee. LG Monitors Silently Install Full-Access Windows App The permissions are what push this beyond a minor annoyance. The app&#8217;s Microsoft Store listing discloses access to &#8220;All system resources&#8221; along with the user&#8217;s internet connection. Notebookcheck, which identified the test unit as an LG UltraGear 34GX900A-B, reports that a McAfee Scam Detector component installs as part of the same package and that LG&#8217;s own installer documentation references data tied to location, hardware configuration, online activity, account credentials, and contacts. In Gamers Nexus&#8217;s testing, McAfee&#8217;s trial offer appeared as the default recommendation in nearly every boot, with occasional promotions for other &#8220;recommended&#8221; LG software mixed in. CyberDigest noted that an internet-connected app with that permission scope could, in theory, rotate in different ads or move data off the machine, though that specific capability was not observed during testing. The distinction matters, and it is also cold comfort, since the underlying problem is that the user never agreed to find out. This is not limited to a single new product line. CyberDigest found the same behavior on newer UltraGear and UltraFine models, on monitors up to three years old, and on units already in daily use at its office, appearing in 31 of 32 boot tests. LG has not issued a public response to the findings. The delivery channel is Windows&#8217; Device Metadata Retrieval Client, which lets hardware vendors tie a Microsoft Store app to a connected device&#8217;s metadata so it installs automatically in the background. That feature was built for drivers, firmware updaters, and companion utilities, not for ad software carrying full-resource permissions, and Microsoft&#8217;s own documentation acknowledges the install generates no user notification. A trusted operating system update mechanism is doing the installing, which is precisely why no one is asked anything. Mitigation Affected users on Windows Pro or Enterprise can block the behavior through Group Policy: run gpedit.msc, navigate to Computer Configuration → Administrative Templates → System → Device Installation, and enable &#8220;Prevent automatic download of applications associated with device metadata&#8221;. Windows Home users lack access to Group Policy Editor and must rely on registry edits or Microsoft Store download settings instead. Users can also check Settings → Apps for the LG Monitor App Installer or McAfee Scam Detector and remove them if already present.
