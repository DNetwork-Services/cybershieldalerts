Anthropic has rolled out Claude Cowork to web and mobile platforms, letting AI agent sessions persist across devices and continue executing tasks even when users step away. The expansion, announced July 7, 2026, marks a shift from desktop-only agentic workflows to a cross-platform system in which work follows the user rather than remaining tethered to a single machine. Claude Cowork lets users delegate multi-step tasks to Claude, which then works across connected files, calendars, email, messaging apps, and web tools until the job is complete. According to Anthropic&#8217;s usage data, more than 90% of Cowork activity is non-coding. Claude Cowork Expands to Web and Mobile Business operations and content creation together account for roughly half of all usage, covering tasks such as reconciling quarterly spend, converting contract folders into renewal trackers with flagged risks, and building client decks from call transcripts. The rollout introduces three functional shifts. First, sessions now offer cross-device continuity: work started on a laptop can be checked from a phone and completed on any device, with files and progress syncing automatically. Second, tasks can now run in the background without an active device connection, so scheduled jobs continue to run even after a laptop is closed. Anthropic illustrates this with a 6 a.m. client-prep task in which Claude reviews email threads and transcripts, builds a briefing document, and drafts a follow-up email for review before the user even checks in. Third, the system preserves human-in-the-loop decision-making. When Claude reaches a point that requires human judgment, it sends a query directly to the user&#8217;s phone, and users can redirect an in-progress task midstream. Anthropic emphasizes that desktop remains the most capable surface, retaining exclusive access to local files and browser integration. The web and mobile expansion is aimed primarily at users who previously couldn&#8217;t run a desktop client at all. Alongside the platform expansion, Anthropic is merging chat and Cowork into a single interface on web and desktop, with projects and artifacts now shared across both surfaces. This change reduces friction between conversational queries and delegated agentic tasks by keeping them within the same session rather than separate tools. According to Anthropic, beta access begins with Claude Max subscribers, with availability expanding to additional plans over the coming weeks. On web, Cowork sessions launch directly from the claude.ai home screen, while on mobile the feature is available through the sidebar in Claude&#8217;s iOS and Android apps. To encourage adoption, Anthropic is also extending doubled Cowork usage limits through August 5, 2026, giving users more room to test larger and more complex tasks during the transition period. For teams managing sensitive data, the background-execution model raises questions that warrant close monitoring. Scheduling tasks without an active device present increases the attack surface for credential exposure and unauthorized task injection if session tokens or connected-app permissions aren&#8217;t tightly scoped. Organizations adopting Cowork across mobile and web should audit which third-party integrations, including email, calendar, and messaging access, are granted persistent permissions, since tasks now execute unattended and outside direct user supervision.
