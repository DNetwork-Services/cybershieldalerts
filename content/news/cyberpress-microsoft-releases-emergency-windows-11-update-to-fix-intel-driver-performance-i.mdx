Microsoft has shipped an out-of-band (OOB) emergency update for Windows 11 to resolve performance regressions tied to an Intel Innovation Platform Framework (Intel IPF) driver. The update, tracked as KB5121767, applies to Windows 11 version 25H2 (OS Build 26200.8894) and Windows 11 version 24H2 (OS Build 26100.8894). Following recent cumulative Windows updates, a limited number of devices running the Intel IPF driver experienced unexpected shifts in system performance, power consumption, and overall behavior. Microsoft Releases Emergency Windows 11 Update The issue affected a subset of Dell devices closely enough that Microsoft temporarily blocked those machines from receiving the July 2026 security update, KB5101650, in order to prevent the regression from spreading further. KB5121767 lifts that block and delivers the targeted fix, allowing previously excluded Dell devices to resume normal update installation. Microsoft has been explicit that this is a targeted fix rather than a universal patch. Devices that are not exhibiting the performance anomaly do not need to install the update, and the company has flagged it as relevant only to the affected subset of hardware. As a cumulative OOB release, KB5121767 includes all prior security and non-security fixes on along with the new Intel IPF correction. The package also includes a servicing stack update, KB5120102 (Build 26100.8872), which improves the reliability of the Windows Update installation process. Alongside this, Microsoft has updated its AI components to version 1.2605.856.0, touching Image Search, Content Extraction, Semantic Analysis, and the Settings Model. This continues Microsoft&#8217;s recent practice of bundling the latest servicing stack update with the latest cumulative update to streamline deployment and reduce installation failures across both managed and consumer devices. Delivery depends on how a device is configured within Windows Update. Devices with &#8220;Get the latest updates as soon as they&#8217;re available&#8221; enabled under Settings > Windows Update will receive KB5121767 automatically without any user action. If that toggle is switched off, users can trigger installation manually by navigating to Settings > Windows Update and selecting Download & Install. The update is currently classified as optional, meaning it can also be found under Settings > Windows Update > Advanced options > Optional updates. For IT administrators and enterprise environments, KB5121767 is also available through the Microsoft Update Catalog, Windows Server Update Services, and the standard Business release channel, enabling staged, controlled deployment across managed fleets. In this case, silent performance degradation on Intel IPF-equipped hardware, particularly affecting OEM devices like Dell&#8217;s lineup, created enough operational risk that Microsoft moved quickly rather than folding the fix into the next rollup. IT administrators managing fleets with Intel IPF drivers, especially Dell hardware previously excluded from KB5101650, should prioritize this update in testing and deployment pipelines. Home users who have noticed unusual power draw, throttling, or behavioral changes since installing the July 2026 security update should check Windows Update immediately.
