F5 released an out-of-band security notification on June 17, 2026, disclosing multiple high and medium-severity vulnerabilities across its NGINX product family. The flaws affect a wide range of products, including NGINX Open Source, NGINX Plus, NGINX Gateway Fabric, NGINX Ingress Controller, and several WAF and DoS protection modules, posing serious risks to organizations running modern web infrastructure. The most critical issues carry CVSS v4.0 scores as high as 9.2, placing them near the top of the severity scale. Critical NGINX Vulnerabilities Fixed CVE-2026-42530 targets the ngx_http_v3_module in NGINX and scores 8.1 (CVSS v3.1) and 9.2 (CVSS v4.0). The vulnerability impacts NGINX Open Source versions 1.31.0–1.31.1, NGINX Instance Manager versions 2.17.0–2.22.0, NGINX Gateway Fabric 1.3.0–1.6.2 and 2.0.0–2.6.3, and NGINX Ingress Controller versions across the 3.x, 4.x, and 5.x release lines. A fix is available only for NGINX Open Source (1.31.2) and NGINX Gateway Fabric (2.6.4); patches for NGINX Instance Manager and Ingress Controller are not yet available. CVE-2026-42055 affects the ngx_http_proxy_v2_module and ngx_http_grpc_module, also scoring 9.2 on CVSS v4.0. This vulnerability has one of the broadest impact surfaces in the advisory, touching NGINX Plus (R33–R36 and 37.0.0–37.0.1), NGINX Open Source 1.30.0–1.31.1, F5 WAF for NGINX, NGINX App Protect WAF, F5 DoS for NGINX, NGINX App Protect DoS, Gateway Fabric, and Ingress Controller. Fixes have been introduced in NGINX Plus 37.0.2.1 and R36 P6, as well as NGINX Open Source 1.31.2 and 1.30.3. Several products, including WAF and DoS modules, have no patches available yet. Two additional high-severity flaws, CVE-2026-11311 and CVE-2026-50107, specifically target NGINX Gateway Fabric versions 2.3.0–2.6.3 and 2.5.0–2.6.3, respectively, both scoring 8.1 (CVSS v3.1) and 8.6 (CVSS v4.0). Both are resolved in NGINX Gateway Fabric version 2.6.4. Two medium-severity issues round out the advisory. CVE-2026-48142 affects the ngx_http_charset_module across a very wide range of NGINX products, scoring 6.3 on CVSS v4.0, with partial patches available. CVE-2026-32682 impacts NGINX Gateway Fabric versions 1.3.0–2.6.3 with a CVSS v4.0 score of 7.1, fixed in version 2.6.4. Mitigation According to F5 advisory, organizations using affected NGINX products should treat this advisory as urgent, particularly given the unpatched status of several critical products. Security teams should: Upgrade NGINX Open Source to 1.31.2 or 1.30.3 immediately Update NGINX Plus to 37.0.2.1 or apply patch R36 P6 Upgrade NGINX Gateway Fabric to 2.6.4 Monitor F5&#8217;s advisory portal for patches for NGINX Instance Manager, Ingress Controller, and WAF/DoS modules Apply network-level controls or WAF rules as compensating measures for unpatched deployments Given NGINX&#8217;s role as one of the world&#8217;s most widely deployed web servers and reverse proxies, organizations should prioritize patching these flaws before threat actors develop working exploits targeting these modules.
