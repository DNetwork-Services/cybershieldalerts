Microsoft will stop delivering OneDrive sync app updates to systems running Windows 10 version 21H2 and earlier on August 15, 2026, creating a new support concern for organizations still operating legacy Windows endpoints. The change was announced in Microsoft 365 Message Center notification MC1426708, published on July 13 and updated on July 17, 2026. After the cutoff date, affected systems will no longer receive enhancements to the OneDrive sync app, bug fixes, or security updates. While the installed OneDrive application may continue to work on unsupported Windows 10 versions, Microsoft warned that future functionality is not guaranteed. Over time, users could encounter degraded synchronization performance, compatibility problems, or failures when using new OneDrive capabilities. OneDrive Sync App Updates for Windows 10 Microsoft said the retirement aligns OneDrive support with its Windows lifecycle policy and enables the company to focus product investments on supported operating systems. The decision affects organizations that use the OneDrive sync client for Microsoft 365 file synchronization on endpoints running Windows 10 21H2 or earlier. Windows 10 version 22H2 remains supported under the new policy. Devices running Windows versions released or later supported will continue to receive OneDrive sync app feature updates, bug fixes, and security patches until October 10, 2028. The security impact is particularly important for enterprise environments. A sync client that no longer receives security updates may become increasingly exposed to vulnerabilities discovered after support ends. Attackers could potentially target unpatched flaws in the application, its authentication workflows, local file handling, or integrations with cloud storage services. Organizations should identify endpoints that are still running Windows 10 21H2 or earlier and prioritize upgrades. Microsoft recommends moving eligible devices to Windows 11. Where a Windows 11 migration is not immediately possible, administrators should upgrade systems to Windows 10 22H2 as a minimum requirement to continue supporting the OneDrive sync app. IT teams should also notify users, service desk personnel, and administrators about the upcoming retirement. Internal documentation should be updated to reflect which Windows versions remain supported for OneDrive synchronization and how users can access corporate files if their device cannot be upgraded immediately. Microsoft confirmed that OneDrive’s web interface will remain available through supported browsers, providing a temporary alternative for unsupported devices, though it does not replace the desktop sync client’s background synchronization. No Microsoft 365 tenant configuration changes are required for this transition. However, organizations should treat the August 2026 deadline as part of their broader endpoint lifecycle and patch-management planning.
