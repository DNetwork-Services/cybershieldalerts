Anthropic has rolled out a new capability in Claude Cowork that lets users teach the AI assistant a repeatable skill simply by recording their screen while performing a task. The feature, called &#8220;Record a skill,&#8221; turns a screen capture into a workflow Claude can execute on demand, eliminating the need to write manual instructions or scripts. The image shared shows the feature in action within the Claude desktop app interface, where users can select &#8220;Record a skill&#8221; from the plus menu. Users start a recording session, perform the task they want Claude to learn, and narrate their actions as they go. Claude&#8217;s underlying system captures screen activity, mouse clicks, keystrokes, and voice commentary, then processes this demonstration into a structured, reusable skill definition. New in Claude Cowork: teach Claude a skill.Record your screen while you do a task, talk through it as you go, and Claude turns it into a skill it can run again. Find it under Record a skill in the + menu of the Claude desktop app. Available on Pro, Max, and Team plans. pic.twitter.com/9OgJLOKWzj&mdash; Claude (@claudeai) July 21, 2026 This approach mirrors the broader Claude Cowork architecture, where a skill functions as a saved set of instructions that Claude can recall and apply to similar tasks in the future. Once captured, the skill appears in the user&#8217;s skill library and can be triggered again without repeating the manual walkthrough. This positions Claude Cowork closer to an &#8220;AI employee&#8221; model, where the assistant learns organizational or personal workflows directly from observed behavior rather than relying solely on text-based prompting. Anthropic&#8217;s broader Cowork toolset already supports local file access, persistent memory, connectors, and Cowork Projects, and this recording-based skill creation extends that automation layer further. Feature Access and Plan Availability The &#8220;Record a skill&#8221; option is accessible directly from the plus (+) menu inside the Claude desktop application, alongside existing options like adding files, browsing skills, connectors, and plugins. Anthropic has confirmed the feature is available exclusively to users on Pro, Max, and Team subscription tiers.
